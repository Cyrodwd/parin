# Examples

You can run every example with:

```d
void main() {
    import popka.examples;
    runEveryExample();
}
```

## [Hello](hello.d)

This example serves as a classic hello-world program, introducing the fundamental structure of a Popka program.

![image](hello.png)

You can run the example with:

```d

}
```
